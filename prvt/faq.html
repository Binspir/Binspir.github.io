English FAQ

Q1:Solution for irregular passcode after updating to Privaut V3.1

Shortcut:update the app to V4.0 onwards, and follow Q3 to reset your passcode

If you don't want to change the passcode, read the following intruction:
The passcode mechanism has changed since V3.0.
Here are a few examples and please try the new ones according to the following old v.s. new patterns
1. old: -0.03141500 -> new:3141500
2. old: 1234+4567, 1234-4567,1234x4567,1234/4567 -> new: try 4567 first. If 4567 doesn't work, try the result of respective calculation
3. old:0.0000... -> new: send an email to support@binspir.com, we will provide you an instruction to reset passcode.

Q2:Data missing after updating to Privault V3.0

Firstly please update to V3.1 onwards. Othewise, please check whether you previously were using an app with a similar icon like ours, since many users mistake another app for ours and have been sending a lot of query emails mistakenly to us.

Steps: Open Appstore app > tap your account icon > tap purchased > browse all the similar apps you used to download and might have been using for a while.

Secondly, if you confirm you are using this app. Please try to relaunch the app, since there is a very rare chance that your data might not be migrated to V3.1 format correctly. Relaunching the app can re-initiate the data migration process. Thirdly, if you were using secondary passcode(previously also known as decoy passcode), your files are still inside the app but you cannot view them for the time being. Please wait for the future release that the data migration feature for this account is available.

if you still have such issue, please send us an email and we will follow-up with you for a recovery solution.

Q3. "What if I forget the passcode?"

1. Update the App to V4.0.
2. Enter "911911" on the keypad and you will see a help list.
3. Tap "Forget Passcode?"
4. Setup and verify an email first if you haven't setup yet
5. Tap "Forget Passcode?" (again) and you will receive a temporary verification code
6. Enter the verification on the keypad to initiate the passcode reset process.

Primero actualice a V3.1 Uno. Además, compruebe si ha utilizado previamente una aplicación similar a nosotros, ya que muchos usuarios confunden otra aplicación con la nuestra y nos envían un gran número de correos electrónicos de consulta por error.

Espere a que se publique en el futuro que la función de migración de datos para esta cuenta esté disponible.

Si aún tiene ese problema, envíenos un correo electrónico y nos pondremos en contacto con usted para encontrar una solución de recuperación.

Q2：PrivaultV3.0にアップデートした後にデータが欠落しています

まず、V3.1以降に更新して、データが表示されているかどうかを再確認してください。 そうでない場合は、多くのユーザーが別のアプリケーションが私たちのものだと勘違いし、クエリーのEメールを誤って送信しているため、以前に私たちと似たアイコンのアプリケーションを使用していたかどうかを確認してください。

まず、以前に私たちのようなアイコンのアプリを使用していたかどうかを確認してください。多くのユーザーが別のアプリを私たちのアプリと間違え、多くのクエリメールを誤って私たちに送信してきました。

手順：Appstoreアプリを開きます>アカウントアイコンをタップします>購入済みをタップします>ダウンロードに使用し、しばらく使用していた可能性のある同様のアプリをすべて参照します。

次に、このアプリを使用していることを確認した場合。データがV3.0形式に正しく移行されない可能性は非常に低いため、アプリを再起動してみてください。アプリを再起動すると、データ移行プロセスを再開できます。

第3に、セカンダリパスコード（以前はおとりパスコードとも呼ばれていました）を使用していた場合、ファイルはアプリ内にありますが、当面は表示できません。このアカウントのデータ移行機能が利用可能になる将来のリリースをお待ちください。

それでも問題が解決しない場合は、メールでご連絡ください。復旧方法についてフォローアップいたします。

Q3. 「パスコードを忘れたらどうなりますか？」

1. アプリケーションをV4.0バージョンにアップグレードする
2. キーパッドに“911911”を入力し、ヘルプリストが表示されます
3. パスコードを忘れて
4. セットアップをして、まだセットアップしていない場合は、最初のメールを確認
5. “パスコードを忘れる？”(そして)あなたは一時的な検証コードを受け取ります
6. パスコードのリセット処理を開始するためにキーパッドの検証を入力します。
